### Quick Deploy Options:

1. **Vercel** (Recommended)
   ```bash
   npm i -g vercel
   vercel
   ```

2. **Netlify**
   - Drag and drop the folder to netlify.com/drop
   - Or use Netlify CLI:
   ```bash
   npm i -g netlify-cli
   netlify deploy
   ```

3. **GitHub Pages**
   - Push to GitHub repository
   - Enable GitHub Pages in repository settings
   - Select main branch as source

4. **Traditional Hosting**
   - Upload all files to your web server via FTP/SFTP
   - Ensure proper file permissions

## ğŸ“ Customization

### Update Personal Information:
- Edit `index.html` for content changes
- Update photos in `assets/images/`
- Modify colors in `styles.css` (CSS variables in `:root`)

### Add New Projects:
1. Add project card in the "Featured Projects" section
2. Use existing project cards as template
3. Update project image/video and description

### Modify Colors:
Edit CSS variables in `styles.css`:
```css
:root {
    --color-accent-blue: #3b82f6;
    --color-purple: #8b5cf6;
    /* ... other colors */
}
```
